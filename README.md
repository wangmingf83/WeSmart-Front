# 微智营前端

#### 介绍
国外的营销触达通道比较单一，主要以EDM为主。国内因为渠道的变化营销场景就复杂很多，当前国内做用户的营销触达主要的场景通道如下：

- 企业微信私聊、社群、朋友圈
- 微信运营公众号、小程序、视频号
- 电商运营抖音、快手、淘宝、京东、拼多多
- 公域运营短信、邮件、APP弹窗、PUSH推送、banner位
- 其他其他渠道
